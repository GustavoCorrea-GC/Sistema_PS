Função que converte uma string em uma estrutura de dados Alarme_t.

Parâmetros
linhaBuffer contendo a string.
AEstrutura de dados Alarme_t em que será armazenada o contêudo da string.
Aviso
A string recebida deverá ter os itens separados por tabs ('\t') e finalizados com um '\n' na seguinte ordem: identificador, N_Serie_Equi, Ativo, Nome, N_Ativacao, Classificacao, DataCadastro(Dia, Mes, Ano),DataEntrada(Dia, Mes, Ano), DataSaida(Dia, Mes, Ano) e Descrição.
